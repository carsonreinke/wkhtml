# WkHtml

Ruby bindings for wkhtmltox (wkhtmltopdf)

## Installation

Add this line to your application's Gemfile:

    gem 'wkhtml'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install wkhtml
